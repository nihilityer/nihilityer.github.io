## 想要阅读的书籍
**做这个的原因**：今天心血来潮决定看一下以前的老番剧，翻来翻去，发现只有少数几个质量很高，而值得二刷的就只剩下了一个《春物》，虽然我没有把这种小说中的哲学当作真理，但是确实给了我一些启发。
并且主角的经历让人看着很舒服。在看番的过程中，突然看到了最终章的一个1星点评（[点评地址]( https://www.bilibili.com/read/cv6889635 )），乍看很有道理，但是我一直信奉一个观点（虽然是自己胡乱总结的），一个作品的上限就是它的作者。
然后心理学虽然发展历史算长，但是说到底还是问题很多，很多东西心理学也争论不休。点评将作品升级到了诠释真理的地步，但我不敢相信真的有人能够解释人际关系这个复杂的命题。
但是这个点评能有这么多认同者，说明小说还是值得我一看的，然后我想起来很久之前我就因为这个原因准备收集原著来拜读一下，奈何现实原因放弃，选择这个现实问题依旧存在，不过现在可以通过这篇文章来记录。

现实中的问题总会解决，只希望因困难阻隔的愿望在条件允许时依旧是我的追求。
