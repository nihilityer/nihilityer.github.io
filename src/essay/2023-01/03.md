## GIT使用注意事项

**查看系统config**

```shell
`git config --system --list　　`
```

**查看当前用户配置**

```shell
`git config --global --list　　`
```

**查看当前仓库配置**

```shell
`git config --``local` `--list`
```

**查看用户邮箱**

```shell
`git config user.email　　`
```

**修改用户邮箱**

```shell
`git config --global --replace-all user.name ``"your user eamil"``【引号里的内容替换成你的邮箱地址】　　　`
```

**查看用户名**

```shell
`git config user.name　　`
```

**修改用户名**

```shell
`git config --global --replace-all user.name ``"your user name"``【引号里的内容替换成你的用户名】`
```
